# MLP do Zero — Projeto

## Introdução

Este repositório contém uma implementação didática de um Multi-Layer Perceptron (MLP) implementado "do zero" usando apenas NumPy para operações matriciais. O objetivo é entender e demonstrar o funcionamento do forward, backward (backpropagation) e treinamento por mini-batch SGD, aplicado ao problema de classificação de dígitos (MNIST).

## Como rodar

1. Criar e ativar um ambiente virtual (recomendado):

```bash
python -m venv .venv
source .venv/bin/activate
```

2. Instalar dependências:

```bash
pip install -r requirements.txt
```
